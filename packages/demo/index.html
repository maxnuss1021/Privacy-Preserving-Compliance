Privacy Preserving Compliance Demo

Three example applications demonstrate the privacy-preserving compliance framework. Each app uses a placeholder ERC-20 token to mimic a real application's integration -- the token's mint function requires a valid zero-knowledge compliance proof before executing. SafeSwap and CleanMixer share the same non-membership ComplianceDefinition (sanction list), so a proof generated for one is reused instantly by the other. VerifiedLend uses a separate membership ComplianceDefinition (whitelist).
